Lista Ordenada

Lista Ordenada é usado para criar Lista que tem uma sequência, como exemplo:
  1. Abrir a Porta do Carro
  2. Entrar no carro
  3. Fechar a Porta
  4. Colocar a Chave no Contato
  5. Ligar o carro
  6. Abrir o portão
  7. Sair com o carro da Garagem
  8. Fechar o portão

Podendo ainda utilizar caracterer do tipo, números, números Romanos, letras maiúsculas e letras Minúsculas

Lista Não Ordenada

Lista Não Ordenada é usado quando a lista a ser feita não será usada em uma sequência, como por exemplo uma lista do mercado:
  • Mamão
  • Limão
  • Aveia
  • Pão
  • Suco
  • Refrigerante
  • Carne

Podemos usar caracteres como: circulos, pontinho preto e quadradinhos
